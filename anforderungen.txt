Wenn die Startseite aufgerufen wird, muss das System eine Übersichtsseite mit einer Liste von Client-Hardware anzeigen.
Nach Auswahl einer Client-Hardware auf der Startseite muss das System dem Benutzer Informationen zur Hardware (CPU, RAM, IP, etc.) sowie zur installierten Software anzeigen.
Wenn die Startseite aufgerufen wird, muss das System dem Benutzer die Möglichkeit bieten, sich einzuloggen.
Nach der Eingabe der Benutzerdaten in die Login-Maske muss das System diese auf Richtigkeit prüfen.
Nachdem der Benutzer sich eingeloggt hat, muss das System eine Übersichtsseite mit einer Liste von Client-Hardware sowie der Möglichkeit, neue Client-Hardware zum System hinzuzufügen, sowie bestehende Client-Hardware zu bearbeiten und zu löschen, anzeigen.
Wenn der Benutzer neue Client-Hardware hinzufügen möchte, muss das System ihm die Möglichkeit bieten, eine Client-Hardware mit bereits installiertem Client-Daemon hinzuzufügen.
Wenn der Benutzer eine Client-Hardware hinzufügen möchte, soll das System ihm die Möglichkeit bieten, auf einer Client-Hardware den Client-Daemon automatisch zu installieren und diese zum System hinzuzufügen.
Wenn der Benutzer eine Client-Hardware auswählt, muss das System dem Benutzer Informationen zur Hardware sowie zur installierten Software anzeigen.
Wenn der Benutzer eine Client-Hardware auswählt, muss das System dem Benutzer die Möglichkeit bieten, neue Software-Pakete aus dem Software-Katalog auf dem Host zu installieren sowie bereits installierte Software zu konfigurieren und zu deinstallieren.
Wenn der Benutzer eine Client-Hardware auswählt, muss das System ihm die Möglichkeit bieten, Dienste auf der Client-Hardware zu starten und zu stoppen.
Nach dem Klick auf einen entsprechenden Menüpunkt muss das System dem eingeloggten Benutzer die Möglichkeit bieten, neue Software-Pakete zum Katalog hinzuzufügen sowie bestehende zu aktualisieren und zu löschen.
Wenn der Benutzer neue Software-Pakete zum Katalog hinzufügen möchte, muss das System das übergebene Paket darauf überprüfen, ob es den Vorgaben zur Paketstruktur entspricht.
